Features

Personal Organizer and CRM

Home Screen:

  • Dashboard with calendar, task list, and quick links to key features

Calendar:

  • Schedule work shifts, appointments, family/social events
  • Integrate with team/group calendars
  • Set reminders and notifications

Tasks:

  • Create tasks with details, priority, deadline, assignee
  • View tasks by status (planned, in progress, completed)
  • Recurring/repeating tasks
  • Reminders for upcoming/overdue tasks

Contacts:

  • Store contact details of other civilians, mentors, team members
  • Groups for family, teams, projects
  • Search and filter contacts
  • Quick call/message integration

Notes:

  • Take notes during meetings, classes, counseling sessions
  • Add attachments like documents, photos
  • Categorize notes by topic, project etc.

Documents:

  • Store certificates, transcripts, ID docs, health records etc.
  • Add file annotations
  • Facilitate document sharing

Journal:

  • Record daily experiences, learnings, thoughts, prayers
  • Private or shared journal entries
  • Journal prompts to track progress over time

Milestones:

  • Track certification progress, career goals, family goals
  • Add target dates and notes
  • Milestone reminders and notifications

CRM

  • Custom message templates
  • Integrations for calls, sms, WhatsApp and Video calls

Additional features:

  • Built-in messaging/chat for communication
  • Maps integration for locations
  • Customizable home screen widgets
  • Backup and restore functionality
  • Private mode for sensitive information
